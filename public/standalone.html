Invoice Generator

How to Use:

1. Enter an invoice number in the field above

2. Click "Generate PDF Invoice"

3. A server-side script will generate the PDF

4. The PDF will be saved in the uploads directory

Generate from the command line:

You can also generate invoices directly from the command line:

node dist/generate-pdf.js [invoice-number]

For example:

node dist/generate-pdf.js 12345

This will generate a PDF at: uploads/invoice-12345.pdf
